四个指标的计算公式
1. 💰 已落袋  (yiLuDai)
适用对象：已退出的用户（代币余额 < 1）
公式：
已落袋 = Σ (卖出总额 - 买入总额)
// 对所有已退出用户求和
yiLuDai += (stats.totalSellSol - stats.totalBuySol)
含义：所有已经清仓用户的实现盈亏总和

2. 🎯 本轮下注 (benLunXiaZhu)
适用对象：当前持有者（代币余额 ≥ 1）
公式：
本轮下注 = Σ (买入总额)
// 对所有当前持有者求和
benLunXiaZhu += stats.totalBuySol
含义：当前所有持有者的买入总投入

3. 💵 本轮成本 (benLunChengBen)
适用对象：当前持有者
公式：
本轮成本 = 本轮下注 - 当前持有者的卖出总额
benLunChengBen = benLunXiaZhu - currentHoldersRealized
详细说明：
本轮下注：当前持有者的买入总额
currentHoldersRealized：当前持有者已经卖出回收的 SOL
结果：当前持有者的净成本（还锁定在代币中的 SOL）
4. 📈 浮盈浮亏 (floatingPnL)
适用对象：当前持有者
公式：
浮盈浮亏 = Σ (当前持有代币市值 - 净成本)
// 对所有当前持有者求和
对于每个持有者：
  当前市值 = 持有代币数量 × 当前价格
  净成本 = netSolSpent (买入花费 - 卖出收入)
  浮盈浮亏 += (当前市值 - 净成本)
